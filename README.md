# Attorney.AI

🏆 **IEM Hackathon Project**

Attorney.AI is a frontend prototype developed during the IEM Hackathon. The project was built to demonstrate the concept of an AI-powered legal assistance platform through an interactive user interface and navigation flow.

The focus of the project was on designing the frontend experience within the hackathon timeframe rather than implementing backend services or AI functionality.

## 📌 Project Overview

The prototype focuses on presenting the user experience and interface design for a legal assistance platform. It was built as a hackathon submission within a limited timeframe.

## ✨ Features

- Responsive homepage
- Multi-page navigation
- User interface prototype
- Clickable navigation between pages

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
